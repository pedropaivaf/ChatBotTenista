# Regras de Codificação - ChatBotTenista

Para garantir o máximo entendimento e fins educacionais, todas as modificações neste projeto devem seguir as seguintes regras:

1.  **Comentários Linha a Linha**: Todo arquivo de código (`.py`, `.js`, `.css`) deve ter comentários explicativos em virtualmente todas as linhas de lógica.
2.  **Linguagem**: Todos os comentários e documentação devem estar em Português (PT-BR).
3.  **Destaques**: Usar termos como "IMPORTANTE", "LÓGICA", "NLTK" nos comentários para destacar partes críticas.
4.  **Manutenção da Arquitetura**: Manter a separação entre `engine.py` (dados técnicos) e `app.py` (servidor/interface).
